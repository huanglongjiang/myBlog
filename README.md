# myBlog
用vue3开发的一个轻盈、简洁的可展示图片、音频、视频和文字！又可以自定义主题及页面特效的的空间博客
![good博客 图片](https://www.theminimalists.com/wp-content/uploads/2019/01/how-to-start-a-blog-in-2020-1.jpg "good博客")

# 默认访问功能
* 首页
* 相册
* 视频
* 音频
* 评论
* 留言
* 友情链接
* 时钟
* 头像
* 个人信息
* 日历
* 置顶
* 主题配置
* 智能主题
* MP3播放器
* 歌词
* 桌面歌词
